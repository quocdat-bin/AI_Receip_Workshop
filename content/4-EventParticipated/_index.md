Throughout my internship at the First Cloud AI Journey (FCAJ) program, I had the opportunity to participate in many meaningful events organized by the program. Each event brought memorable experiences, helping me access a lot of new, useful, and practical knowledge. Especially, among them, 3 events left a profound impression on me, not only because of the valuable lessons but also because of the memorable moments and meaningful gifts the program dedicated to the interns.

### [Event 1](4.1-Event1/)
&emsp;**Event Name:** AWS FIRST CLOUD AI JOURNEY COMMUNITY DAY

&emsp;**Date & Time:** 9:00 AM, Saturday, May 23, 2026 

&emsp;**Location:** 26th Floor, Bitexco Tower, No. 02 Hai Trieu Street, Sai Gon Ward, Ho Chi Minh City 

&emsp;**Role:** Attendee 

&emsp;**Event Description:** This is an event for the IT community aimed at updating the latest trends in Cloud Computing and Generative AI. The sharing session focused on practical solutions such as designing specialized AI Agent systems, optimizing the interaction process with LLMs, and network infrastructure security techniques on the AWS platform for enterprises.
