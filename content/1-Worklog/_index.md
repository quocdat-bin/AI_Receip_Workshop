On this page, I summarize in detail my **Worklog** recording my entire graduation internship process at **Amazon Web Services Vietnam Co., Ltd. (AWS)**.

The internship program was conducted and completed over **12 weeks** (from April 17, 2026 to July 09, 2026), closely linked to the development of the **AI Recipe Finder** food recommendation system. Below is a summary of the main tasks performed each week:

**Week 1:** [Learn about the First Cloud AI Journey program, register an AWS account, complete 5 account tasks to claim credits, and set up AWS Budgets](1.1-week1/)

**Week 2:** [Set up MFA authentication to secure the account, study FCJ bootcamp modules on YouTube, and complete the Amazon EC2 creation lab](1.2-week2/)

**Week 3:** [Hold group meetings to propose the AI Recipe Finder project, analyze expected AWS services, and practice static web hosting with Amazon S3](1.3-week3/)

**Week 4:** [Research access management with IAM Groups/Users and basic permissions, explore Amazon RDS, and design AWS architecture on draw.io](1.4-week4/)

**Week 5:** [Research the AWS Bedrock service and submit requests for Service Quotas for the AI model](1.5-week5/)

**Week 6:** [Search for suitable datasets, research Amazon RDS, and attend the FCAJ Community Day event](1.6-week6/)

**Week 7:** [Launch EC2 instances, import data into Amazon RDS, test EC2 to RDS connections, and troubleshoot deployment issues](1.7-week7/)

**Week 8:** [Learn about Elastic IP addresses and Amazon API Gateway, and configure AWS Bedrock to integrate with the web application](1.8-week8/)

**Week 9:** [Review application features, coordinate system integration testing, audit AWS resources, and handle Bedrock quota revocation by transitioning to new solutions](1.9-week9/)

**Week 10:** [Read technical blogs on AWS Study Group VN, update project workflow based on mentor feedback, and switch to Google AI API](1.10-week10/)

**Week 11:** [Conduct pre-summary system review, prepare project demo content, and clone the report template for customization](1.11-week11/)

**Week 12:** [Complete the final presentation workshop, submit materials to the portal, and finalize the Internship Progress Report](1.12-week12/)
